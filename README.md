## ⚙️ How to Use

1. Clone or download the repository.
2. Open HTML files directly in your browser.
3. Run Java files using any Java IDE or terminal.
4. For Bootstrap files, ensure internet access for CDN or use local assets.
5. For jQuery exercises:
   1. Open the .html files in a modern browser.
   2. Ensure internet connection for loading jQuery from CDN or use the included local jQuery files.
6. For ANSI SQL exercises:
   1. Use MySQL CLI or MySQL Workbench to run .sql scripts.
   2. Connect to a MySQL server instance with proper privileges.
   3. Run scripts sequentially to understand database setup, querying, and advanced features.
7. For jQuery exercises:
   1. Open the C# project/solution in Visual Studio or your preferred IDE.
   2. Configure your database connection string in appsettings.json or code.
   3. Build and run the sample files to test different ADO.NET concepts.
   4. For Entity Framework examples, ensure database migrations are applied.
